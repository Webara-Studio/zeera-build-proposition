# Zeera Ghana Build Proposition

Webara Studio's presentation for the Zeera Ghana sportsbook build proposition.

## Contents

- `index.html` - tablet-friendly presentation with keyboard navigation and print styles
- `output/Zeera_Webara_Build_Proposition.pdf` - A4 landscape booklet/print version

## Local preview

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Deployment

This is a static presentation and can be imported directly into Vercel. No build command or environment variables are required.
